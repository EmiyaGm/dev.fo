Emitter 插件使用

emitter 简介

emitter 是一个监听事件的插件，可以方便的拿到交易的事件，并按照自己的需求存储区块上的数据。

使用 emitter 插件的优势

  • 不受类似与 mongo 插件的约束，能够按照自己的业务需求存储数据。
  • 相比较 histroy 插件占用内存小。

结语

区块上的数据不仅仅可以保存于 MySql 数据库，还能同步保存于 db模块 下支持的其他数据库。

例如：

  • sqlite
  • mongodb 注意emitter 不支持 mongodb， 因为支持了 eos 的 mongodb 插件, FIBJS 禁用了 mongodb 插件。
  • leveldb
  • Redis
